# 扫码打开小程序

## 前言

为了方便和推广小程序的使用，微信公众平台支持扫描普通链接二维码跳转小程序能力。

## 需求背景

项目：一款蓝牙设备遥控器小程序
需求：根据链接二维码内容添加至遥控器列表、显示对应遥控器界面
服务器：云服务

## 配置流程

1. 登录小程序后台，进入“开发管理-开发设置-扫普通链接二维码打开小程序”开启配置
2. 服务器地址，域名需要通过ICP备案
3. 下载校验文件保存至服务器，一般存在根目录
4. 添加二维码配置填入域名、页面地址、测试范围
  - 开发环境
  - 体验环境
  - 线上环境
5. 发布线上

## 云开发配置流程

1. 下载校验文件上传至云开发存储根目录
2. 点击上传文件查看并复制地址
3. 小程序后台添加二维码配置
4. ...
