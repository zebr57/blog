## node 到底用来做什么的？

### node 改变了前端的开发方式

1. 为前端提供开发工具（vue-cli，webpack 等）
2. 提供线上环境模拟（开启本地服务）
3. 提供中间层服务渲染（SSR、高流量项目）
4. 做后端服务（极少）

### node 核心操作

1. 读取文件
2. 开启服务
